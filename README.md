# CSC2053Project
This is a school group Project

Summary:
Project Definition
Goal: Use bootstrap to create and style a calendar tool to assist students manage their schedules and stay organized.

Core functionalities:

1)	Event Creation:   
•	Allow users to create events with titles, descriptions, dates, and times.  Add options for repeating events (daily, weekly, and monthly).  
•	Allows you to schedule dedicated events such as club meetings, sports practices, or social gatherings in addition to academic obligations.

2)	Calendar Views:   
•	Button to change calendar views (week, month).  Make it easy for users to switch between perspectives. Users may quickly examine their schedule and make plans by flipping between the week and month displays with ease.

3)	Todo List:
•	Provide a todo list with checkable tasks. By using the to-do list tool to keep track of assignments, appointments, and other crucial chores, users can effortlessly maintain organization. Users may utilize this feature to prioritize their chores and mark them as finished.
